# Gemini Terminal Add-on

This add-on provides a web-based terminal with the Gemini CLI pre-installed, allowing you to interact with Google's Gemini AI from your Home Assistant dashboard.

## Installation

Follow the instructions in the [Home Assistant documentation](https://www.home-assistant.io/hassio/installing_third_party_addons/) to add this repository to your Home Assistant instance.
